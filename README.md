# ReflexiveRL

**A Research Framework for Endogenous and Reflexive Reinforcement Learning**

---

## Overview

ReflexiveRL is an autonomous research-grade framework for studying **reinforcement learning in endogenous environments**, where predictive signals generated by the agent influence the system dynamics themselves. 

The framework moves beyond exogenous Markovian assumptions:
\[
s_{t+1} \sim P(s_{t+1} \mid s_t, a_t)
\]

to model **coupled, prediction-dependent evolution**:
\[
s_{t+1} = f(s_t, a_t, \rho_\theta(s_t), \eta)
\]

This induces non-stationary effective dynamics, shifting the learning step from an optimization problem under fixed dynamics to the **stabilization of a coupled dynamical system**.

---

## Core Novelty: Why ReflexiveRL?

ReflexiveRL explicitly addresses the **Observer Effect** in machine learning.
- **Standard RL (e.g., OpenAI Gym, Stable Baselines)** treats the environment as a black-box transition $P$ that responds only to actions.
- **ReflexiveRL** treats the environment as a system whose very physics are modified by the agent's internal predictions ($\rho$).

**Key Distinction**: In ReflexiveRL, the transition function is effectively policy-dependent through the oracle, making the environment non-stationary with respect to the learning system. Gradients propagate not just through $R$, but through the causal pathways of the transition $f$.

## Technical Proofs & Empirical Validation

The repository includes a suite of scripts designed to validate the theoretical core of reflexive learning.

### 1. Endogenous Gradient Proof
- **Script**: `scripts/prove_endogenous_gradients.jl`
- **Result**: Demonstrates that differentiating through the transition $f$ using Zygote corrects for the **prediction bias** that standard policy gradient methods ignore. Endogenous PG achieves significantly lower RMSE to convergence targets in coupled environments.

### 2. Systematic Ablation Study
- **Script**: `scripts/ablation_study.jl`
- **Validation**: Compares Classical RL vs. Oracle-Only vs. Full Reflexive.
- **Outcome**: Proves that while simple oracles can predict $r_p$, only the combination of **endogenous feedback** and **stability-aware loss** prevents divergence at high coupling strengths ($\alpha > 1.0$).

### 3. Phase Transition Analysis
- **Script**: `scripts/phase_transition_analysis.jl`
- **Mapping**: Identification of three distinct stability regimes:
    - **Convergent**: Passive stability where $r_p$ tracks local fixed points.
    - **Oscillatory**: Limit cycle behavior where predictions induce overcorrection.
    - **Divergent**: Feedback amplification leading to system explosion.

## Core Concepts

### Reflexive Operator (Theoretical Foundation)
The system is formalized as a coupled operator:
\[
\Phi(s) = T(s, \pi(\rho(s)))
\]
Learning corresponds to identifying approximate fixed points $s^* \approx \Phi(s^*)$ by minimizing the fixed-point residual:
\[
\| s - \Phi(s) \| \approx \mathbb{E}[\| s_t - s_{t+1} \|]
\]

### Endogenous Dynamics
Transition mechanics: $s_{t+1} = f(s_t, a_t, r_p), \quad r_p = \rho_\theta(s_t)$.

## Algorithms Implemented

### Endogenous Gradient Projection (EGP)
- **Path**: `src/algorithms/egp.jl`
- **Optimization**: $\mathcal{L} = -\mathbb{E}[R] + \beta_{stab} \cdot \mathbb{E}[(s_{t+1} - s_t)^2]$
- **Note**: Approximates full endogenous gradients via stability-aware regularization.

### Fixed-Point Reinforcement Learning (FPRL)
- **Path**: `src/algorithms/fprl.jl`
- **Optimization**: $\mathcal{L} = \mathcal{L}_{PG} + \lambda_{fp} \cdot \mathbb{E}[(s_t - s_{t+1})^2]$
- **Note**: Encourages convergence to low-residual fixed points.

---

## Evaluation Metrics

- **Reflexive Consistency**: $\mathbb{E}[\| s_t - s_{t+1} \|]$ (Fixed-point residual).
- **Feedback Sensitivity (Proxy)**: $\Delta s = f(s, a(r_p), r_p) - f(s, a(0), 0)$.
- **Stability Regime Indicator**: Automated detection of Convergent/Oscillatory/Divergent states.

- **Feedback Amplification**: Scenarios where the agent's oracle reinforces stochastic noise.
